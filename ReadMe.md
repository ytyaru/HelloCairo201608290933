# このソフトウェアについて #

HelloGtkPlus201608252332は私個人が学習目的で作成したソフトウェアです。

Cairoを使って文字列を描画しました。

# 開発環境 #

* Windows XP SP3
* Visual C++ 2010 Express
    * C++/Win32

## ライブラリ ##

* [glade3-3.6.7-with-GTK+.exe](http://ftp.gnome.org/pub/GNOME/binaries/win32/glade3/3.6/glade3-3.6.7-with-GTK+.exe)
    * GTK+ 2.16.2.1602
    * cairo 1.8.6
